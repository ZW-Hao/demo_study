尺寸和重量

长度 :      162.9mm


宽度 :      75.5 mm


厚度 :      玻璃版：9.1 mm； 素皮版：9.5 mm


重量 :      玻璃版：约 212 克（含电池）； 素皮版：约 212 克（含电池）


*实际尺寸与重量依配置、制造工艺、测量方法的不同可能有所差异。




屏幕

尺寸 :      6.76 英寸


*显示屏采用圆角设计，按照标准矩形测量时，屏幕的对角线长度是 6.76 英寸（实际可视区域略小）。


色彩 :      1670 万色，DCI-P3 广色域


类型 :      OLED，最高支持 90 Hz 刷新率、240 Hz 触控采样率。


分辨率 :     FHD+ 2772 x 1344 像素    *该分辨率对应标准矩形，实际屏幕有效像素略少。




处理器

麒麟 9000


CPU :    八核，1 x Cortex-A77@3.13 GHz + 3 x Cortex-A77@2.54 GHz + 4 x Cortex-A55@2.05 GHz


GPU :      24 核 Mali-G78


NPU :      双大核 NPU + 微核 NPU (神经网络处理单元)。






操作系统

操作系统 :      EMUI 11.0（基于Android 10）
